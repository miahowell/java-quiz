## Java Student Quiz & Management System

## Description
This project is a Java-based application designed to simulate a quiz and classroom management system.
It allows users to log in as either a student or a teacher, with different functionality based on their role.

The system demonstrates object-oriented programming principles, file handling, and role-based access control.

## Features

Student Functionality
- Secure login using username and password
- Automatically generates a quiz with 10 random questions
- Questions are selected from a pool of 150+ stored in a file
- Displays final score upon completion

Teacher Functionality
- Login with elevated permissions
- Add new quiz questions to the system
- Delete existing questions
- View student scores and performance

## Technologies Used
- Java
- File I/O (reading and writing data to files)
- Object-Oriented Programming (OOP)

## How to Run
1. Clone or download this repository
2. Open the project in an IDE (such as NetBeans, Intellij, or VS Code)
3. Locate the main class file
4. Run the program

## Project Structure
- Main Class – controls program flow and user interaction
- User Classes – handles student and teacher roles
- Quiz Logic – manages question selection and scoring
- File Handling – reads questions and stores updates

## Key Concepts Demonstrated
- Object-oriented design (classes, methods, encapsulation)
- Role-based system functionality
- Randomized data selection
- File handling for persistent data storage
- User input handling and validation
